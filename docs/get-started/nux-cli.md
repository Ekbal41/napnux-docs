# Napnux Cli

You can you napnux cli to easily generate a project for you.

```bash
npm install nux-cli
nux-cli init
```

This will create a basic projrct in the given directory.

### Project Structure

```bash
.
├── apps
│   └── notes
│       ├── views
│       │   └── welcome.html
│       ├── public
│       ├── styles
│       └── index.js
│
├── root.js
├── server.js
├── package.json

```
